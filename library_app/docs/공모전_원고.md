# 데이터로 진단하고 처방하는 도서관 장서
### — 자기참조 수급(需給) 격차 분석 기반 자동 진단·큐레이션 모델

> 공모 부문: ① 도서관의 새로운 역할에 맞는 신규 서비스/기술 제안 · 현안 해결 모델 개발
> (작업용 초안 — 최종본은 A4 15매 이내로 HWP/Word 변환)

---

## 1. 문제 정의 — 왜 지금 이 문제인가

도서관 예산은 한정되어 있고, 한 번 구입한 책은 서가·인력·공간이라는 지속 비용을 낳는다. 그러나 *무엇을 더 사고(수서), 무엇을 정리할지(장서 점검)* 를 정하는 의사결정은 여전히 담당자의 경험과 직관에 크게 의존한다. 그 결과 두 가지 현안이 반복된다.

- **수요-공급 불일치**: 이용자가 원하는 분야의 책은 늘 부족하고, 정작 서가의 상당 부분은 거의 대출되지 않는 책이 차지한다.
- **데이터 기반 의사결정의 부재**: 도서관은 대출 로그라는 풍부한 데이터를 매일 쌓지만, 이를 *수서·장서관리 결정*으로 환원하는 도구가 없다.

본 제안은 이 현안을 **"도서관 데이터로 장서를 자동 진단하고, 처방(큐레이션/수서)까지 연결하는 모델"** 로 해결한다. 핵심 질문은 하나다 — *이 도서관의 장서는, 이 도서관 이용자의 수요와 얼마나 어긋나 있는가?*

## 2. 방법론 — 자기참조 수급 격차 모델 (핵심 차별점)

### 2.1 왜 '절대 기준'이 아니라 '자기참조'인가

장서를 진단하는 가장 흔한 방식은 절대 기준이다(예: "청소년 장서는 전체의 10% 이상이어야 한다"). 그러나 도서관은 저마다 역할이 다르다. 어린이도서관이 아동 장서 비중이 높은 것은 *문제가 아니라 정상*이다. 절대 기준은 이런 맥락을 무시해 잘못된 진단을 낳는다.

본 모델은 외부 기준 대신 **그 도서관 내부의 두 분포를 비교**한다.

- **공급(供給)** = 장서가 어떻게 구성되어 있는가 → `도서권수`
- **수요(需要)** = 실제로 무엇이 대출되는가 → `대출건수`

두 분포의 **비중 격차**가 곧 의사결정 신호다.

| 격차 | 의미 | 처방 |
|---|---|---|
| 수요비중 > 공급비중 | 원하는데 책이 부족 | **보강(수서) 후보** |
| 공급비중 > 수요비중 | 책은 많은데 안 빌려감 | **정리/큐레이션 후보** |

이 자기참조 방식은 도서관 종류가 달라도 공정하게 작동한다는 결정적 장점이 있다. (보조 지표로 전국 인기대출 분포를 '바깥 수요'의 벤치마크로 활용해 확장할 수 있다.)

### 2.2 데이터와 진단 절차

- **데이터**: 도서관정보나루(data4library.kr) — 도서관별 장서/대출 데이터(주제분류·도서권수·대출건수·부가기호 등). 본 분석은 대구 2·28도서관 장서 대출목록(장서 59,438권)을 사례로 사용한다.
- **세그먼트화**: 주제분류번호(KDC) 맨 앞자리로 *분야* 대분류, ISBN 부가기호 맨 앞자리로 *독자대상*(교양·실용·청소년·아동·전문 등)을 도출한다.
- **자동 진단**: 세그먼트별 공급·수요 비중과 회전율(대출÷장서)을 계산하고, 비중 격차가 임계값(±3%p)을 넘으면 사서의 판단 규칙을 코드화한 **진단 문장을 자동 생성**한다. 이는 별도 학습이 필요한 AI가 아니라 *해석 가능한 규칙 기반 모델*로, 도서관 현장이 신뢰·검증하기 쉽다.

## 3. 분석 결과 — 2·28도서관 사례

전체 회전율은 4.72회(책 1권당 평균 대출 횟수)다. 세그먼트별로 들어가면 평균 뒤에 숨은 불균형이 드러난다.

### 3.1 독자대상별 — 헤드라인: '아동 수요'와 '교양 장서'의 어긋남

| 독자대상 | 공급비중 | 수요비중 | 격차(%p) | 회전율 |
|---|---|---|---|---|
| **아동** | 29.5% | **56.7%** | **+27.2** | **9.08** |
| 실용 | 7.7% | 8.5% | +0.8 | 5.16 |
| 청소년 | 2.6% | 1.3% | −1.3 | 2.36 |
| 전문 | 8.1% | 1.3% | −6.8 | 0.76 |
| **교양** | **45.5%** | 25.3% | **−20.2** | 2.62 |

이 도서관 대출의 **절반 이상(56.7%)이 아동 도서**이며 회전율은 9.08회로 압도적이다. 그러나 아동 장서는 전체의 29.5%에 그친다. 반대로 장서의 45.5%를 차지하는 교양서는 수요가 25.3%에 머물고, 전문서는 회전율 0.76회로 사실상 서가에 머무른다. → **자연어 진단(자동 생성):** *"아동 도서는 수요 대비 장서가 크게 부족(보강 1순위)하고, 교양·전문서는 장서 과잉(정리·큐레이션 대상)이다."*

### 3.2 분야별(KDC)

| 분야 | 공급비중 | 수요비중 | 격차(%p) | 회전율 |
|---|---|---|---|---|
| **자연과학** | 5.8% | 12.6% | **+6.8** | **10.35** |
| 문학 | 36.0% | 40.7% | +4.7 | 5.33 |
| 예술 | 5.5% | 2.5% | −3.0 | 2.13 |
| **사회과학** | 22.3% | 14.1% | **−8.2** | 2.99 |

자연과학(회전율 10.35)과 문학은 수요 대비 장서가 부족해 보강이 필요하고, 사회과학은 장서 22.3%로 가장 많지만 수요는 14.1%에 그쳐 과잉 신호를 보인다.

### 3.3 사회적 함의 — 진단이 곧 '포용성' 점검

수급 격차 진단은 운영 효율을 넘어 **누구를 위한 장서인가**를 드러낸다. 아동 수요가 폭발하는 이 도서관에서 청소년 장서(2.6%)는 빈약하고, 회전율이 낮은 전문서가 예산을 묶어둔다. 모델은 이렇게 *취약하거나 소외된 독자층을 자동으로 가시화*하여, 한정된 예산을 더 포용적으로 배분하는 근거를 제공한다. [보강: 전국 평균 또는 인구통계 대비 비교를 더하면 '격차'의 사회적 크기를 정량화할 수 있다.]

## 4. 제안 모델 — '진단 → 처방' 자동화 + 범용성

본 모델은 단일 분석이 아니라 **재사용 가능한 시스템**으로 설계한다.

1. **자동 진단**: 도서관 데이터를 입력하면 위 수급 격차·회전율을 계산해 진단 문장을 생성한다. *(프로토타입 구현 완료 — `diagnosis.py`)*
2. **처방(큐레이션/수서 추천)**: 진단에서 '부족'으로 판정된 세그먼트에 대해, 전국 인기대출 도서 중 **자관 미소장 도서**를 추천한다(인기대출·소장여부 API 결합). [보강: 다음 구현 단계]
3. **범용 확장**: data4library의 `libCode` 파라미터로 **어느 도서관이든 데이터를 자동 수집·진단**할 수 있어, 전국 단위로 확장 가능하다. [보강: API 연동 단계]

산출물은 **시각화 대시보드**(웹) 형태로, 담당자가 분야·독자대상별 격차를 한눈에 보고 추천 목록을 받는다.

## 5. 기대 효과

- **데이터 기반 예산 배분**: 직관이 아닌 수급 격차 근거로 수서·정리 우선순위를 정한다.
- **포용적 장서 운영**: 소외 독자층을 자동 포착해 형평성 있는 장서 구성을 유도한다.
- **전국 확장성**: `libCode` 기반 범용 모델로, 인력·데이터 역량이 부족한 중소 도서관도 동일 진단을 받을 수 있다 — 도서관 간 정보 격차 완화.

## 6. 한계와 향후 과제

- 대출건수는 등록 이후 *누적*이고 도서권수는 *현재* 시점이라 시간 기준이 일부 다르다 → 등록일·발행연도 보정으로 정밀화 가능.
- 신착 도서(분석 시점 대출 0건 약 34%)가 회전율을 낮추는 효과 → 신착 보정 필요.
- 프로그램 신청 방식·이용자 접근성 등은 본 데이터로 직접 측정 불가 → 외부 통계·서비스 설계로 보완.
